# AI Ad Copywriter

An AI-powered Ad Copywriter that helps marketers, founders, agencies, and businesses generate high-converting advertising copy in seconds.

The application uses OpenAI models to analyze product information, identify customer pain points, generate marketing angles, and create ad copy for multiple platforms including Facebook, Google, Instagram, LinkedIn, and TikTok.

---

## Features

### Product Analysis
- Analyze product descriptions
- Identify key benefits
- Extract unique selling points (USPs)

### Audience Research
- Define target audience
- Generate customer personas
- Identify customer pain points

### Copy Generation
Generate:

- Headlines
- Hooks
- Ad Copy
- CTAs (Call-To-Actions)
- Product Descriptions
- Landing Page Copy

### Multiple Frameworks
Supports popular copywriting frameworks:

- AIDA (Attention, Interest, Desire, Action)
- PAS (Problem, Agitate, Solution)
- BAB (Before, After, Bridge)
- FAB (Features, Advantages, Benefits)

### Multi-Platform Support
Generate optimized copy for:

- Facebook Ads
- Google Ads
- Instagram Ads
- LinkedIn Ads
- TikTok Ads
- X (Twitter) Ads

### Export Options
- Copy JSON
- Download TXT
- Export PDF
- Save History

### AI-Powered Suggestions
- Marketing Angles
- Emotional Triggers
- Value Propositions
- CTA Recommendations
